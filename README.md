StoreStats
==========

An API to track purchases, bounce rate, customer geographic, and more in real-time using node.js and socket.io

Stack Used:
-----------

* node.js
* socket.io
* express.js
* SASS / Compass
* CSS3 flexbox
* Require.js / Optimization tools
* AngularJS
* Highcharts.js
~~ * Handlebars.js ~~

live example coming soon on appfog.com integrated with mongodb
